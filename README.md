# HealthRing - 智能戒指女性健康AI系统

一款专为 20-45 岁育龄女性设计的智能健康监测应用，结合智能戒指数据，提供精准的健康追踪、周期管理、AI健康建议和健康预警。

## 🎯 产品定位

以**24小时无感佩戴**为基础，通过核心生理数据采集与算法分析，为用户提供通俗化、精准化的健康状态监测与预警，打造轻量化个人健康管理终端。

## 👥 目标用户

涵盖女性人生不同阶段的健康需求：

- **避孕需求用户（22-40岁）** - 精准识别安全期/易孕期，辅助自然避孕
- **备孕需求用户（25-35岁）** - 精准预测排卵日，提升受孕成功率
- **孕期需求用户（25-38岁）** - 监测孕期健康，保障母婴安全
- **健康管理达人（20-45岁）** - 多维度数据追踪，优化生活方式

## ✨ 核心功能

### 🔍 精准生理监测
- **心率监测** - 实时心率、静息心率、心率变异性（HRV）
- **体温监测** - 基础体温、皮肤温度连续追踪
- **睡眠分析** - 睡眠分期、睡眠质量评分、睡眠规律评估
- **压力管理** - 压力指数、自主神经平衡状态
- **活动追踪** - 活动强度、步数、卡路里消耗

### 📅 女性周期管理
- **精准周期预测** - 基于生理数据的排卵日和经期预测
- **多模式支持** - 普通模式、备孕模式、孕期模式自由切换
- **安全期/易孕期识别** - 结合体温、HRV等多维度数据
- **周期健康建议** - 针对不同周期阶段的个性化指导

### 🚨 健康智能预警
- **感染早期预警** - 提前捕捉感染苗头（体温微升、心率异常、HRV骤降）
- **心律失常筛查** - 心动过速/过缓、心律不齐初步识别
- **头痛预警** - 偏头痛、紧张性头痛前兆识别
- **疲劳度监测** - 身体疲劳状态评估与提醒

### 🤖 AI健康助手
- **基于数据的智能问答** - 结合真实生理数据的个性化回答
- **主动健康提醒** - 久坐、喝水、放松、睡眠建议
- **健康方案定制** - 作息、运动、放松方案推荐
- **每日健康贴士** - 可执行的小动作建议

### 🎯 目标管理
- **多维度目标** - 活动、睡眠、体征等各类健康目标
- **实时进度追踪** - 首页卡片式进度展示
- **智能复盘优化** - 未达标原因分析、目标难度推荐
- **勋章激励体系** - 连续达标、突破记录成就

## 🚀 快速开始

### 环境要求

- Node.js 18+
- pnpm (推荐) 或 npm / yarn

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm dev
```

应用将在 http://localhost:3000 启动

### 构建生产版本

```bash
pnpm build
```

### 启动生产服务器

```bash
pnpm start
```

### 代码检查

```bash
pnpm lint
```

## 📱 功能模式

### 普通模式（默认）
面向日常健康监测，展示核心健康数据、戒指状态、常用功能入口。

### 备孕模式
面向有备孕计划用户，主打体温监测、排卵期预测、经期管理。

### 孕期模式
面向已怀孕用户，主打孕周管理、孕期健康监测、产检提醒。
